# Movimento do Cavalo em C

Projeto desenvolvido para a disciplina de programação em C.

O programa simula o movimento do Cavalo no xadrez utilizando loops aninhados.

## Movimento implementado

O Cavalo realiza o seguinte movimento em "L":

- Baixo
- Baixo
- Esquerda

## Conceitos utilizados

- Linguagem C
- Loop `for`
- Loop `while`
- Loops aninhados
- `printf`

## Compilação

```bash
gcc xadrez.c -o xadrez
